## Usage

Get the files.

```
$ git clone https://github.com/AlexanderSelzer/staticcharge.git
$ cd staticcharge
```

Install the dependencies.

```
$ npm install -g gulp
$ npm install && cd app && bower install && cd ..
```

Be ready to go

```
$ gulp dev
```

Gulp is watching your changes, and will automatically recompile.

#### Have fun :)
